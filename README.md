# PointBlank_TargetShootingVerilogProject
This project implements a Target shooting game like at carnivals called ‘Point Blank’. It is  implemented on the Basys-3 FPGA board. The primary objective of the system is to  stimulate a real-time interactive game in which the player shoots down multiple targets  within a limited time frame.


- Codes can be found in 'pointblankattempt.srcs' 
- You will have to add the .coe files again to the project with proper folder path 
- images + files + .py converter is attached
